# Simple Omelette Recipe Page

A responsive recipe webpage built using **HTML and CSS**. This project focuses on structuring content semantically and styling it to create a clean and readable recipe layout.

## Overview

This project is a solution to a challenge from **Frontend Mentor**. The goal was to build a visually appealing recipe card layout that displays:

- Recipe image
- Preparation time
- Ingredients list
- Step-by-step instructions
- Nutrition table

The layout is designed with **readability and clean structure** in mind.

## Built With

- **HTML5** – semantic structure
- **CSS3**
- **Flexbox**
- **Custom fonts**
- **Responsive layout techniques**

## Features

- Clean recipe card layout
- Responsive image
- Styled lists with custom markers
- Structured instruction steps
- Nutrition information displayed using a table
- Custom fonts using `@font-face`
- Soft color palette for better readability

## What I Learned

While building this project I practiced:

- Writing **semantic HTML**
- Structuring content using **sections and articles**
- Styling lists and markers using `::marker`
- Creating **tables with proper spacing**
- Using **Flexbox for layout alignment**
- Loading **custom fonts locally with `@font-face`**
- Creating reusable color variables and consistent typography
